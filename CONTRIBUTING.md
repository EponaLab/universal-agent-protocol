# Contributing to Universal Agent Protocol

Thank you for your interest in contributing! This is a community-driven proposal.

## Ways to Contribute

### 1. Feedback & Discussion

- **Open Issues**: For bugs, unclear sections, or suggestions
- **Discussions**: For broader conversation and ideas
- **Pull Requests**: For concrete improvements

### 2. Documentation

Help improve:
- Clarify confusing sections
- Add more examples
- Fix typos or errors
- Improve diagrams

### 3. Implementation

Build proof-of-concepts:
- Reference implementations
- Example integrations
- Testing tools
- SDK extensions

### 4. Advocacy

Spread the word:
- Share in communities
- Write blog posts
- Present at meetups
- Build demos

## Guidelines

### For Issues

- **Search first**: Check if already reported
- **Be specific**: Include context and use cases
- **Be constructive**: Propose solutions when possible

### For Pull Requests

- **Small & focused**: One improvement per PR
- **Explain why**: Include rationale in description
- **Follow style**: Match existing formatting
- **Test examples**: Ensure code examples work

### For Discussions

- **Be respectful**: Assume good intent
- **Stay on topic**: Keep focused on protocol design
- **Share experience**: Real-world insights are valuable

## Code of Conduct

- Be respectful and inclusive
- Critique ideas, not people
- Assume good faith
- Focus on what's best for the community

## Questions?

Open an issue with the "question" label.
